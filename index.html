CQEngine

简介

CQEngine是一款开源的集合查询引擎，有了CQEngine，我们可以使用SQL-LIKE的方式进行java集合的查询。

CQEngine可以进行毫秒级查询。相应速度微妙级。

CQEngine中有index的概念，可以给不同的属性字段添加index，在查询时可以大大加快速度。

支持嵌套查询。

备注

CQEngine只针对查询做优化，查询后的结果可以转为stream操作获取到数据。
针对大量数据，stream转集合实际上会比普通stream慢很多。
CQEngine主要是针对查询做的优化，在查询时可以进行排序，但是重新排序意味着数据整体结构的改变，添加排序会花相对较多的时间。

atomic

简介

为解决高并发情况下线程安全问题，synchronized使用悲观锁效率低。
原子操作 线程安全

CAS

乐观锁策略
compare and swap

简单讲就是如果想要替换的数据不是现在的数据吗，则拒绝替换
即原始值为A，想将A替换为B时，原始值已经变成了C，这意味着已经有程序改动了原始值，则不进行值替换

disruptor使用CAS算法不是用锁解决并发问题
需要的朋友可以去查看一下

数据库

简介

正常做项目我们可以选择使用
mybatis, JPA

mybatis

通过注解或者xml文件方式将java对象和sql语句映射成sql
通过框架进行执行，并返回最终结果。

JPA

实现对象/数据库表映射框架
支持使用原生sql，或者直接使用方法名进行数据库操作，涉及到更新操作时使用事务

底层使用EntityManager进行持久化

spring 事务

在spring中使用事务需要在不同类调用方法时开启
或者通过代理拿到proxy对象
同级方法调用，及其调用的方法有开启事务的注解，注解也不会生效

git server

简介

我们使用github时候，不花钱的话只能公开(最近可以将项目private)
于是我们便有了搭建git服务器的需求

git 服务器

git其实只是一款应用
我们可以在服务器上直接安装

安装后我们需要设置连接方式
可以使用无需密码的ssh连接方式

重点是需要将本地公钥传到服务器上，即id_rsa.pub
我们可以使用git bash在本地生成ssh key
然后将公钥写入服务器git中的authorized_keys

过程类似github项目使用ssh方式上传，详细之后有时间再更新

项目

服务器上创建一个项目
在本地项目中添加远程地址
即可以提交到服务器中的git项目上，供多人合作

ssh

我们可以通过生成ssh的方式拉取远程代码库的代码
我们可以ssh-key保存到windows或其他凭据中，下一次再认证是不需要重复输入密码

备注

在搭建git服务器同时，也可以再搭建一波wiki
