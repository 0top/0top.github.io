POI

简述

poi即Poor Obfuscation Implementatio
poi是Apache软件基金会的开放源码函式库
java程序开发者可以使用poi提供的api读取Microsoft Office文件
以下内容会简述excel读取相关的一些内容

poi

1.格式

HSSF - 可以读取.xls文件
XSSF - 可以读取.xlsx文件

2.poi模型

poi将excel中的模型都定义到org.apache.poi.ss.usermodel包中
比如workbook，sheet，cell等
通过操纵这些模型对应excel
我们可以创建或者解析excel文件

3.读取

excel读取重点在于解析cell表格中的数据

excel表格中的单元格由不同的单元格格式具有不同的dataformate
但单元格内容决定因素是单元格中的值

cell中值七种形式:
_NONE、BLANK、BOOLEAN、ERROR、FORMULA、NUMERIC、STRING
首先对于numeric类型的cell，我们可以读取日历和数字格式

注意点

1.参考文档中的一句话：
If what you want to do is get a String value for your numeric cell, stop!. This is not the way to do it. Instead, for fetching the string value of a numeric or boolean or date cell, use DataFormatter instead.

2.对于excel中拆分已合并的单元格：
excel中合并单元格都将会将值填入最右上角的单元格中
所以可以先将合并的单元格全部取出，然后记录右上角的cell

注意，如果批量拆分单元格。注意循环时每次释放合并的单元格时
