git server

简介

我们使用github时候，不花钱的话只能公开
于是我们便有了搭建git服务器的需求

git 服务器

git其实只是一款应用
我们可以在服务器上直接安装

安装后我们需要设置连接方式
可以使用无需密码的ssh连接方式

重点是需要将本地公钥传到服务器上，即id_rsa.pub
我们可以使用git bash在本地生成ssh key
然后将公钥写入服务器git中的authorized_keys

过程类似github项目使用ssh方式上传，详细之后有时间再更新

项目

服务器上创建一个项目
在本地项目中添加远程地址
即可以提交到服务器中的git项目上，供多人合作

备注

我们可以搭建git服务器
也可以再搭建一波wiki
